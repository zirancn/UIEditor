UIEditor by zhangziran
=============================================================

比葫芦娃更好用的前端界面调试工具类
=============================================================

置入方式：
任意阶段注册一个parent给UIEditor即可
=============================================================

操作说明：
拖动类名/按方向箭/WSAD 调整位置（shift更快）
Q/E 选中当前选中对象的parent/child
R 退出编辑模式
ESC 选中状态下可强制取消选中
鼠标滚轮 伸缩（shift 5倍速）
X 切换横/纵向伸缩
Z 可回到初始状态
=============================================================

特别说明：
非选中状态可使用游戏的快捷捷
快捷点失效的话请结束编辑重新开始
=============================================================
